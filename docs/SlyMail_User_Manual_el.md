# Εγχειρίδιο Χρήστη SlyMail

Αυτό το αρχείο υπάρχει ώστε το εγχειρίδιο να είναι διαθέσιμο σε όλα τα υποστηριζόμενα από το πρόγραμμα γλωσσικά σύνολα (όπως στο Musicians Canvas).

**Σημείωση**: Η πλήρης μετάφραση δεν έχει ολοκληρωθεί ακόμη. Για το πλήρες περιεχόμενο, δείτε την αγγλική έκδοση:

- `SlyMail_User_Manual.md`

Οι εκδόσεις HTML/PDF/TXT θα παραχθούν από αυτό το αρχείο όταν ολοκληρωθεί η μετάφραση.
